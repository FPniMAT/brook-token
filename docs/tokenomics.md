# Token Economics

This document outlines the economic model of the **brook-token** ecosystem, including token supply, issuance, distribution, governance, and lifecycle management. A well-defined token economy ensures long-term sustainability, transparency, and alignment between users, developers, and stakeholders.

---

# Purpose

The purpose of the tokenomics model is to:

- Establish a transparent token supply.
- Define how tokens are created and distributed.
- Support sustainable ecosystem growth.
- Incentivize network participation.
- Maintain long-term economic stability.
- Provide governance opportunities for token holders.

---

Brook Token is an ERC-20 compatible digital asset designed for use across decentralized applications and blockchain ecosystems. The token may be used for payments, governance, staking, rewards, liquidity provisioning, and other ecosystem utilities.

# Total Supply

The total token supply represents the maximum number of tokens that can exist within the ecosystem.

Possible supply models include:

- Fixed supply
- Capped supply
- Inflationary supply
- Dynamic supply

The current implementation determines whether additional tokens can be minted after deployment.

---

# Minting

Minting refers to the creation of new tokens.

If enabled, minting may be restricted to authorized accounts through ownership or role-based access control.

Potential use cases include:

- Ecosystem incentives
- Treasury funding
- Staking rewards
- Community grants
- Liquidity incentives

---

# Burning

Token burning permanently removes tokens from circulation.

Burning mechanisms may be used to:

- Reduce circulating supply
- Increase token scarcity
- Offset inflation
- Support long-term value

Burned tokens cannot be recovered.

# Vesting

Token vesting prevents large allocations from entering circulation immediately.

Common vesting schedules include:

- Team allocations
- Advisor allocations
- Strategic investors
- Treasury releases

Vesting promotes long-term commitment and reduces short-term market pressure.
